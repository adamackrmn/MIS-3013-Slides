Variables & Expressions

Every application you will write, no matter how big or how small, will always use these two concepts. We will be learning how to create/initialize, assign and update variables.

Variables

Variables are used to store values of various types.

Values could come from :
  • The user of the application
  • A database
  • A file / web-service
  • Something that you derived within the application
There are two types of variables, Primitive, sometimes referred to as Simple, and Complex

Primitive Data Types

Type .NET Type Size
bool System.Boolean 0.125-bytes (1-bit)
char System.Char 2-bytes (16-bits)
int System.Int32 4-bytes (32-bits)
double System.Double 8-bytes (64-bits)

8 bits = 1 byte

Microsoft documentation about Simple Data Types

Integer vs. Floating Point

  • Integers are great for storing whole numbers.

  • Floating Points are great for storing numbers that need precision.

Constant Variables

Constant variables are for when you have a value that is not dynamic (will not change throughout the application).

We declare these values as constants because ...
  • Not a literal constant
  • Adds meta-information
  • More flexible

Example: Our business grows and we go from having a 3 day work week to 5, and maybe eventually 7

Create a constant variable to hold how many inches are in a meter. 
                    
                            static void Main(string[] args)
                            {
                                const double inchesInAMeter = 3.28084 * 12; // ft in a meter * inch in a ft
                                const double feetInAMeter = 3.28084;
                                const double inchesInAMeter = feetInAMeter * 12;
                            }

Output

Hello World! - Everyone should start off writing the Hello World application. Display the text on the screen 
                    
                            static void Main(string[] args)
                            {
                                Console.WriteLine("Hello World!");
                            }
'Hello World!' is a series of characters, so we must wrap it with double quotation mark's. This forms a series of individual characters into a string

Input from the Console

Ask the user for their name and then greet them! 
                    
                            static void Main(string[] args)
                            {
                                Console.WriteLine("What is your name? >>");
                                string name = Console.ReadLine();
                                Console.WriteLine("Hello " + name + ", nice to meet you!");
                                Console.WriteLine($"Hello {name}, nice to meet you!");
                            }
To get input from the user via the Console, we use the Console class and call the ReadLine method.
The ReadLine method will give us the value of whatever the user typed in, until they hit the return/enter key, and give it to us in the format of a string. We assign this string value to a variable, 'name', where we can reference it later.

Input from a File

Get all the contents of the file 'Readme.txt' and output it to the console. 
                    
                            using System.IO;

                            static void Main(string[] args)
                            {
                                string allContentsOfFile = File.ReadAllText("Readme.txt");
                                Console.WriteLine(allContentsOfFile);
                            }
